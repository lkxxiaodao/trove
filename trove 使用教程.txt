1. 简介 — trove 是什么

trove 是一款 Windows 桌面效率工具，集三大功能于一身：

  • 剪贴板历史 — 自动记录你复制过的文本、文件和图片，随时回溯复用
  • 微笔记卡片 — 随手记录灵感、待办、知识片段，图文混排，可悬浮桌面
  • 本地定时任务 — 定时弹窗提醒、打开程序/文件、运行脚本

特点：完全离线运行，数据存储在本地 SQLite 数据库，不上传任何信息。
没有账号、没有云同步，你的数据只属于你自己。

2. 快速上手

2.1 安装

  方式一（推荐）：下载安装包，双击安装，自动创建桌面快捷方式。
  方式二（源码）：安装 Python 3.12+，执行 pip install -r requirements.txt，
                 然后 python main.py 即可运行。

2.2 首次启动

  启动后，trove 会在系统托盘显示图标。
  主窗口左侧有四个导航项：

    剪贴板历史  — 查看和管理复制过的内容
    微笔记      — 管理你的笔记卡片
    定时助手    — 管理定时任务
    设置        — 调整偏好与快捷键

  点击导航项即可切换页面。

2.3 关闭与后台运行

  默认情况下，点击窗口 ✕ 按钮不会退出程序，而是隐藏到系统托盘继续运行。
  这样剪贴板监听和定时任务都不会中断。
  如需完全退出：右键托盘图标 → 选择"退出"。

3. 剪贴板历史 (ClipCache)

3.1 它能帮你做什么

  每次 Ctrl+C 复制的内容自动存入历史列表，包括：
    • 纯文本 — 文字、代码、链接等
    • 文件路径 — 在资源管理器中复制的文件
    • 图片 — 截图工具、画图板等复制的图片

  从此不再"刚才复制的内容去哪了？"

3.2 找回复制过的内容

  打开主窗口 → 剪贴板历史页面，所有历史记录按时间倒序排列。
  双击任意条目，其内容即被重新写入剪贴板，直接粘贴到目标位置即可。

3.3 搜索历史

  页面顶部的搜索框支持实时过滤。输入关键词，列表即时缩小范围。
  搜代码片段、搜链接、搜电话号码，随搜随得。

3.4 星标收藏

  重要的剪贴内容，点击条目右侧的 ★ 按钮即可收藏。
  点击搜索框旁的「星标」按钮，仅显示已收藏条目。
  星标条目不会被自动清理，长期保留。

3.5 转为笔记

  右键某条剪贴记录 → 选择「转为笔记」，
  内容自动创建为一张新笔记卡片，保留原文本。

3.6 正则删除（批量清理）

  工具栏的「正则删除」按钮可批量删除匹配特定模式的条目。
  例如：
    • \d{15,19}      — 删除疑似银行卡号/身份证号的内容
    • https?://[^\s]+ — 删除所有网址
    • .{200,}        — 删除超长内容（200 字符以上）

  操作前可预览匹配结果，确认后再执行，避免误删。

3.7 隐私过滤

  在设置中可添加正则过滤规则，匹配的剪贴内容将被自动丢弃，不存入历史。
  适合过滤密码、密钥等敏感信息。参见第 14 节。

3.8 自动清理

  默认最多保存 1000 条记录（可调范围 100～50000），超出后自动删除最旧的非星标条目。
  可设置"自动删除超过 N 天的记录"，如 30 天后自动清理。

4. 快速粘贴

4.1 什么时候用

  不想打开主窗口翻历史，想在当前输入位置快速选择并粘贴历史内容。

4.2 怎么用

  在任意应用的输入区域按 Ctrl+Shift+V（默认，可自定义）：

    ① 弹窗出现在光标附近，列出最近 50 条剪贴历史
    ② 鼠标点击或方向键选择目标条目
    ③ 按 Enter 确认 → 内容自动粘贴到前台应用
    ④ 按 Esc 取消

  支持文本、文件和图片三种类型的粘贴。

4.3 拖拽弹窗

  弹窗顶部的标题栏可以按住鼠标左键拖拽，把弹窗移到不挡视线的地方。
  点击窗口外任意位置，弹窗自动关闭。

──────────────────────────

5. 写笔记 (NoteNest)

5.1 它能帮你做什么

  NoteNest 以卡片式布局管理你的碎片笔记，适合：
    • 随手记录灵感和想法
    • 收藏代码片段、命令行、参考链接
    • 写读书笔记、会议纪要
    • 图文混排的知识卡片

  笔记页面顶部有三个子标签页：普通笔记 | 任务笔记 | 回收站。

5.2 新建笔记

  方式一：在笔记页面点击「+ 新建笔记」，填写内容后保存。
  方式二：全局快捷键 Ctrl+Alt+N（默认，可自定义），在任意窗口弹出新建对话框。

  编辑器中可以：
    • 填写标题（必填）
    • 撰写正文，支持富文本排版
    • 插入图片：点击「🖼 插入图片」按钮，选择本地文件
      （支持 png / jpg / gif / bmp / webp）
    • 设置背景颜色和字体颜色（详见第 6 节）
    • 添加标签（最多 5 字）

5.3 编辑笔记

  双击卡片，或右键选择编辑，打开编辑器即可修改。

5.4 排序方式

  点击工具栏的「排序: 时间」按钮切换模式：
    • 时间排序 — 按最近修改时间从新到旧排列（默认）
    • 自定义排序 — 手动拖拽卡片到任意位置，松手即保存新顺序

5.5 批量操作

  点击工具栏「批量选择」进入批量模式：
    ✓ 勾选多张卡片
    ✓ 一键设置颜色 / 标签
    ✓ 一键删除

5.6 搜索笔记

  工具栏搜索框输入关键词，实时过滤标题和正文匹配的笔记。

6. 笔记的颜色与标签

6.1 8 种预设配色

  新建或编辑笔记时可选择背景色和字体色，8 种预设一目了然：

    名称    背景色        字体色      适合场景
    默认    #FFFFFF       #000000     日常记录
    黄色    #FFF9C4       #333333     高亮/重要提醒
    绿色    #C8E6C9       #1B5E20     完成/正面内容
    蓝色    #BBDEFB       #0D47A1     信息/参考
    粉色    #F8BBD0       #880E4F     个人/情感
    紫色    #E1BEE7       #4A148C     创意/灵感
    灰色    #F5F5F5       #424242     归档/低优先级
    橙色    #FFE0B2       #BF360C     待办/行动项

  选中的颜色按钮会显示蓝色实线边框，直观指示当前选择。

6.2 自定义颜色

  • 背景色：点击 🎨 按钮，从系统取色器中任意挑选
  • 字体色：点击右下角的字体色预览方块，从取色器中挑选
  • 修改预设字体色：点击任意预设颜色下方的小圆点，单独更改该预设的字体颜色

6.3 标签筛选

  • 页面顶部标签下拉框：选择"工作"、"学习"、"待办"等标签，仅显示对应笔记
  • 笔记页面顶部的 8 个预设颜色圆点：点击快速筛选该颜色的笔记
  • ⌄ 按钮：弹出自定义颜色色盘，筛选使用过自定义颜色的笔记
  • 再次点击取消筛选，或点击「全部」恢复全部显示

7. 任务清单

7.1 什么是任务笔记

  任务笔记是一种特殊的笔记类型，内容区每行一个勾选项，点击方框即可标记完成。
  适合：
    • 购物清单
    • 每日待办
    • 项目检查清单
    • 步骤操作指南

7.2 创建任务笔记

  方式一：切换到「任务笔记」标签页 → 点击「+ 新建笔记」
          → 默认已勾选「设为任务笔记」
  方式二：在普通笔记编辑器中，勾选「设为任务笔记（每条一行，可勾选完成）」

  内容区每行输入一个任务项，保存后自动转为带勾选框的清单。
  注意：任务笔记仅支持纯文本，不可插入图片。

7.3 勾选完成

  • 点击方框 → ☑ 打勾 + 文字变灰 + 删除线
  • 页面和悬浮窗中的勾选状态实时同步
  • 可随时取消勾选，恢复未完成状态

7.4 普通笔记 ↔ 任务笔记互转

  编辑笔记时勾选/取消「设为任务笔记」即可切换：
    • 普通 → 任务：内容按行分割为任务项
    • 任务 → 普通：任务项合并为纯文本，勾选状态丢失

7.5 定时弹出任务笔记

  编辑任务笔记时，可设置定时弹出规则：
    • 不启用 — 仅手动打开
    • 一次性 — 指定日期时间弹出一次
    • 每天 — 每天固定时间弹出
    • 每周 — 指定星期几 + 时分弹出

  还可勾选「开机时自动打开此任务笔记」，每次启动 trove 自动弹出。

8. 悬浮笔记 & 幽灵模式

8.1 什么是悬浮笔记

  点击笔记卡片上的悬浮按钮，笔记会变成无边框、始终置顶的浮动窗口，
  钉在桌面最上层，不随其他窗口切换而隐藏。

  适合：
    • 写作参考：参考资料悬浮在文档窗口上方
    • 待办清单：任务笔记悬浮在桌面，随时查看勾选
    • 代码片段：常用命令悬浮在终端上方

8.2 普通笔记悬浮窗操作

  标题栏区域：
    • 拖拽移动：按住标题栏拖动到任意位置
    • 调整大小：拖拽右下角自由缩放
    • 切换颜色：点击标题栏预设颜色圆点，或 🎨 取色器自定义背景色
    • 透明度调节：点击 − / + 按钮调整窗口透明度（20%～100%）
    • 锁/解锁：左下角锁图标切换只读/可编辑状态
      - 锁定后内容不可编辑，防止误触
      - 解锁后可自由编辑（500ms 防抖自动保存）
    • 关闭悬浮：点击右上角 ✕

  内容区右键菜单：
    • 编辑 — 打开完整编辑对话框
    • 更改字体颜色 — 单独调整字体颜色（不影响背景色）
    • 幽灵模式 — 切换鼠标穿透模式
    • 取消悬浮 — 将笔记还原为普通卡片

8.3 幽灵模式

  幽灵模式让悬浮窗"鼠标穿透"——点击悬浮窗就像直接点到它下面的窗口。
  笔记仍然可见，但不影响你对下层窗口的操作。

  开启：在悬浮窗内容区右键 → 勾选「幽灵模式」
  退出：
    • 悬浮窗右键 → 取消勾选「幽灵模式」
    • 笔记页面幽灵管理栏 → 点击对应 chip 的 ✕
    • 快捷键 Ctrl+Alt+G（默认，可自定义）→ 退出所有幽灵模式悬浮窗

  幽灵模式运行时，笔记页面的标签筛选栏下方会出现「👻 幽灵模式中」管理栏，
  列出当前所有幽灵模式悬浮窗，可逐个退出或全部退出。

8.4 任务笔记悬浮窗

  与普通笔记悬浮窗类似，额外支持：
    • 可交互勾选框 — 点击直接标记完成
    • 页面和悬浮窗中的勾选状态实时同步

8.5 自动恢复

  退出 trove 时，当前正在悬浮的笔记会被记住。
  下次启动时自动恢复到桌面原位。

9. 回收站

9.1 软删除机制

  删除的笔记不会立即永久消失，而是先移入回收站。
  误删了？去回收站找回来。

9.2 操作

  • 切换到「回收站」标签页查看所有已删除笔记
  • 勾选要恢复的笔记 → 点击「恢复选中」→ 回到原列表
  • 点击「清空回收站」永久删除所有回收站内容（不可逆，请谨慎）

10. 全局搜索

10.1 一键搜遍所有模块

  按 Ctrl+Alt+V（默认，可自定义），弹出浮动搜索窗口。
  输入关键词，同时搜索：
    • 剪贴板历史 — 匹配文本内容
    • 笔记 — 匹配标题和正文
    • 定时任务 — 匹配任务名和描述

10.2 结果与跳转

  搜索结果按三个模块分组展示。双击某条结果，主窗口自动跳转到对应页面。

10.3 操作技巧

  • 拖拽移动：按住搜索图标区域可拖拽窗口
  • 关闭搜索：按 Esc 或点击右上角 ✕
  • 搜索防抖 200ms，输入流畅不卡顿
  • 也可在主窗口左侧点击搜索图标打开

11. 定时任务 (TaskFlow)

11.1 它能帮你做什么

  TaskFlow 是一个完全离线的定时调度引擎。设置好时间和动作，到时自动执行。
  典型场景：
    • 定时提醒：每小时喝水、每天 18:00 写日报、每周五下午开会
    • 自动打开：每天 9:00 打开工作文档、项目文件夹
    • 运行脚本：定时备份、定时清理（需确认后执行）

11.2 新建任务

  切换到「定时助手」页面 → 点击「+ 新建任务」：

  ① 填写任务名称（必填）和描述（选填）
  ② 选择时间规则：

     规则类型      格式示例                    说明
     一次性        2026-06-05 14:30           指定日期时间执行一次
     每天          09:00                      每天固定时间执行
     每周          周一、周三、周五 @ 09:00     指定星期几执行
     每月          15 号 @ 09:00              每月指定日期执行
     每隔N天       每 3 天 @ 09:00            每隔 N 天执行一次

     提示：每月选"最后一天"时，仅当月存在该日期时触发；
           如选 31 号，则仅在 1/3/5/7/8/10/12 月触发。

  ③ 设置日期范围（可选）：指定任务的开始日期和结束日期

  ④ 选择触发动作（可多选）：

     ☑ 弹窗提醒    — 输入文字和插入图片，到时间弹出图文混排提醒窗口
     ☑ 打开软件    — 启动指定的 exe 程序（可能需要管理员权限）
     ☑ 打开文件    — 打开文档、表格等文件
     ☑ 打开文件夹  — 用资源管理器打开指定目录
     ☑ 运行脚本    — 执行 bat/ps1/py 等脚本（执行前弹确认框）

  ⑤ 可选提示音：
     • 系统提示音（默认）
     • 或选择自定义音频文件（wav / mp3 / flac / ogg）

  ⑥ 保存后任务立即开始调度。

11.3 提醒弹窗

  到达设定时间，弹出无边框置顶提醒窗口：
    • 显示任务名称和弹窗内容（文字+图片）
    • 支持拖拽移动和自由缩放，图片随窗口宽度自适应
    • 点击「知道了」或按 Esc 关闭

11.4 管理任务

  • 启用/禁用：列表中点击「启用」或「禁用」按钮控制任务开关
  • 编辑：双击任务行，修改规则或动作
  • 删除：点击「删除」按钮
  • 查看日志：点击「日志」按钮，展开最近 30 条执行记录
    （✓ 成功 / ✗ 失败，含时间戳和详细信息）

11.5 状态筛选

  页面顶部下拉框可切换视图：
    • 全部 — 显示所有任务
    • 进行中 — 已启用且未过期
    • 已过期 — 超过结束日期
    • 已停用 — 手动禁用的任务

──────────────────────────

12. 备份与恢复

12.1 自动备份

  trove 默认每 30 分钟自动备份一次三个数据库（剪贴板、笔记、任务）。
  备份文件保存在 %APPDATA%/trove/backups/ 目录。
  备份间隔和保留版本数可在设置中调整（间隔 5～1440 分钟，版本 1～50）。

12.2 手动备份

  打开设置 → 点击「立即备份」，将当前三个数据库复制到备份目录。

12.3 恢复数据

  打开设置 → 点击「恢复数据」→ 选择之前的备份文件。
  恢复前会自动先备份当前数据，防止覆盖误操作。
  恢复后需要重启软件生效。

13. 界面主题

trove 支持 8 种 Material Design 主题，在设置页面的下拉框中选择即可实时切换：

  浅色系              深色系

14. 快捷键与设置

14.1 全局快捷键

  以下快捷键在任何应用中均可使用（可在设置中自定义修改）：

    功能               默认快捷键        用途
    ──────────────────────────────────────────────────
    全局搜索           Ctrl+Alt+V       唤出跨模块浮动搜索窗口
    快速新建笔记       Ctrl+Alt+N       弹出新建笔记对话框
    快速粘贴           Ctrl+Shift+V     弹出粘贴历史选择弹窗
    退出幽灵模式       Ctrl+Alt+G       一键退出所有幽灵模式悬浮窗

  修改快捷键：打开设置 → 在「全局热键」区域点击要修改的快捷键 → 按下新组合键。
  编辑热键时自动暂停全局热键，防止快捷键冲突。

14.2 剪贴板设置

  • 最大条数：100～50000，默认 1000
  • 自动删除超过 N 天的记录：默认 0（禁用），可设置 1～365 天
    注意：这与数量上限独立运作 —— 即使总数未达上限，超期记录也会被清理

14.3 备份设置

  • 自动备份间隔：5～1440 分钟，默认 30 分钟
  • 保留版本数：1～50，默认 5 个版本
  • 备份位置：可自定义备份目录（默认 %APPDATA%/trove/backups/）

14.4 外观与行为

  • 关闭窗口时隐藏到系统托盘：开启后 ✕ 按钮不会退出程序
  • 开机时自动启动：通过注册表设置，trove 随系统启动

14.5 系统托盘

  右键托盘图标提供的菜单：
    • 显示/隐藏 — 切换主窗口可见性
    • 新建笔记 — 快速创建笔记
    • 退出 — 完全关闭程序

  双击托盘图标也可显示/隐藏主窗口。

14.6 隐私过滤

  在设置中可管理正则过滤规则，匹配的剪贴板内容将被自动丢弃。
  支持增删规则，规则保存后即时生效。

  提示与技巧

  • 剪贴板内容 MD5 去重：重复复制同一内容不会重复存储，
    而是将已有条目提升到列表顶部，保持列表整洁。

  • 悬浮窗 + 幽灵模式 = 桌面参考面板：
    将参考资料悬浮→开启幽灵模式→随心点击下方窗口不受影响。

  • 任务笔记 + 定时弹出 = 每日清单：
    创建每日待办任务笔记→设置每天 9:00 弹出→开机自动打开。

  • 快速粘贴 + 星标 = 常用文本快捷输入：
    星标常用回复模板/地址/账号→快速粘贴弹窗中星标条目排前面。

  • 遇到问题？
    - 检查系统托盘图标，确保 trove 后台正在运行
    - 查看日志：%APPDATA%/trove/logs/app.log
    - 数据目录：%APPDATA%/trove/data/ （三个数据库文件）
